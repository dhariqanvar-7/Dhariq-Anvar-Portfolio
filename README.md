# Dhariq Anvar J Portfolio

A responsive personal portfolio built mostly with HTML and CSS, with only a small amount of JavaScript.

## Files

- `index.html` — page structure and content
- `css/style.css` — complete design and responsive layout
- `js/script.js` — mobile menu and simple role-changing text
- `assets/resume/` — place your résumé PDF here

## How to run

Open `index.html` in a browser.

For a better development workflow, open the folder in VS Code and use the **Live Server** extension.

## Before publishing

1. Replace the GitHub and LinkedIn placeholder links in `index.html`.
2. Replace `#` in the project buttons with real repository and demo links.
3. Add your résumé as:
   `assets/resume/Dhariq-Anvar-J-Resume.pdf`
4. Update education or project information if needed.
5. Optionally replace the initials avatar with your photograph.

## JavaScript used

JavaScript is intentionally minimal:

- Opens and closes the mobile navigation.
- Changes the role text every 2.2 seconds.

Everything else is built with HTML and CSS.

## Profile photo

Replace `assets/images/profile-placeholder.svg` with your own image. You may keep the same filename or update the image path in `index.html`.

## Project images

Local illustrations are included for Street Food Finder, Tour Booking Management System, and Choice Based Credit System.
